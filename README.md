# Vue Typescript Boilerplate

![logo](https://repository-images.githubusercontent.com/264248082/9d0bf700-96dc-11ea-9d22-ccca962eb56c)

- Vue-CLI 3
- Typescript
- Sass/Scss
- ESLint

## Getting Started

Clone project and enter in folder:

```
cd vue-typescript-boilerplate
```

Run npm install:

```
npm i
```

Enjoy:

```
npm run serve
```
